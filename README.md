# 🚀 Machine Learning Internship Projects

Welcome to my **Machine Learning Internship Portfolio**. This repository contains multiple real-world projects completed as part of the internship program at **Future Interns**.

## 📄 TASK 1: Sales Forecasting
- **Goal**: Predict future sales using time series analysis
- **Model**: SARIMA, Prophet
- **Results**: 92% accuracy, RMSE: 245.67
- **Tech**: Python, pandas, statsmodels, Prophet

## 📋 Task 02: Customer Support Classification
- **Goal**: Auto-classify support tickets into categories
- **Model**: Random Forest + TF-IDF
- **Results**: 87.5% accuracy, F1-Score: 0.85
- **Tech**: NLP, NLTK, scikit-learn, XGBoost

## 📄 TASK 3: Resume Screening System
- **Goal**: Screen & rank resumes for HR roles
- **Features**: Skill extraction, TF-IDF similarity, gap analysis
- **Results**: 2,484 resumes processed, Top score: 80.8%
- **Top Skill Gaps**: Paychex (99.4%), PHR (99.2%), Employment Law (99.1%)
- **Tech**: NLP, scikit-learn, pandas, matplotlib
